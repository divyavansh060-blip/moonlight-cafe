# 🌙 Moonlight Cafe - Premium Mobile WebApp

A modern Premium Mobile Food Ordering Web Application built using HTML, CSS and JavaScript.

The application provides a smooth mobile experience with shopping cart, favorites, coupon system, WhatsApp ordering, UPI payment support, multilingual interface and Progressive Web App (PWA) features.

---

# ✨ Features

## 🍔 Food Ordering

- Premium Mobile UI
- Beautiful Hero Banner
- Trending Dishes
- Category Filters
- Live Search
- Add to Cart
- Quantity Increase/Decrease
- Dynamic Cart Badge

---

- GST Calculation
- Delivery Charges
- Empty Cart Screen

## 🎁 Coupon System

Coupon Code:

```

```

Features

- ₹50 Discount
- Minimum Order ₹150
- Auto Remove if order becomes below ₹150
